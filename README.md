# OOP# Object-Oriented Programming Assignment

## Description
A Python program demonstrating:
- Class design (Superhero base class)
- Inheritance (FlyingHero, TeleportingHero, SwimmingHero)
- Polymorphism (different `move()` implementations)
- Encapsulation (protected `_secret_base` attribute)

## Features
1. **Base Class**: `Superhero` with:
   - Constructor (`__init__`)
   - Methods (`introduce`, `power_up`, `move`)
   - Encapsulated attribute (`_secret_base`)

2. **Inherited Classes**: Each with unique `move()` behavior
3. **Polymorphism**: Single interface (`move()`) with multiple forms
4. **Encapsulation**: Controlled access to secret base
